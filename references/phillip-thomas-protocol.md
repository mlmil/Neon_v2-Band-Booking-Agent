# Phillip Thomas Protocol

Phillip Thomas is an external Santa Barbara booking contact.

## Rule

Never commit Neon Blonde to a date, pay rate, or venue through Phillip without Mike's explicit approval.

## Drafting Style

Keep messages professional, friendly, and concise.

Include only confirmed facts:
- available dates
- band size
- set length
- basic equipment needs
- contact path for follow-up

Mark unknowns as TBD rather than inventing details.

## Availability Responses

Before offering dates:
1. Check Band Sheet confirmed gigs.
2. Check the Neon Blonde calendar for member outs.
3. Apply the two-pass availability verification rule.
4. Offer only dates that pass the confidence gate.

If a date is promising but uncertain, say what needs Mike's confirmation.

## Telegram

`@Neonbandman_bot` routes through the Hermes gateway. Do not build a separate polling script.

Mike's Telegram chat ID is stored in the gateway configuration; do not expose it unless needed for debugging.
